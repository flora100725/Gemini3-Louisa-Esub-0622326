Samsung Medison HERA Z20 診斷用超音波影像系統 510(k) 審查報告
一、 行政摘要 (Executive Summary)
<span style="color:#F87171">本案產品 Samsung Medison HERA Z20 是一套先進的診斷用超音波影像系統，其主要設計目的在於獲取高品質的人體超音波影像並進行體液影像分析。該產品已於美國 FDA 取得 510(k) 上市許可（案號：K241971），其預期用途涵蓋了極其廣泛的臨床領域，包含胎兒/產科、腹部、婦科、手術中應用、小兒科、小器官、新生兒頭部、成人頭部、經直腸、經陰道、肌肉骨骼、泌尿科、成人及小兒心臟、胸部、經食道心臟以及周邊血管影像。本報告旨在針對該產品申請台灣查驗登記之技術文件進行深度審查，參考依據包括美國 FDA 2023 年發布之診斷用超音波指引、TFDA 臨床前測試基準、軟體確效指引、網路安全指引以及人工智慧/機器學習 (AI/ML) 技術指引。</span>

<span style="color:#F87171">根據初步審查紀錄，本案在電性安全 (IEC 60601-1:2025) 與電磁相容 (IEC 60601-1-2:2014) 方面已檢附 Nemko Korea 之合格報告，且軟體確效與網路安全評估亦具備初步完整性。然而，本案在「超音波設備特定安全要求 (IEC 60601-2-37)」、「探頭生物相容性報告」、「臨床測量準確度資料」以及最為關鍵的「AI CADe/CADx 軟體技術文件」方面存在顯著缺失。特別是針對 AI 功能如 ViewAssist、HeartAssist 及 S-Detect 等，原廠尚未能提供符合 TFDA AI/ML 技術指引要求的演算法架構、數據集多樣性證明及獨立性能驗證數據。本報告將針對上述缺陷進行深度的差異分析，並提出具體的補件策略與技術要求，以確保該醫材之安全性、有效性與性能符合國際標準及我國法規。</span>

二、 產品技術規格與操作模式分析
<span style="color:#F87171">HERA Z20 系統採用尖端的數位波束合成 (Digital Beamforming) 與脈衝都卜勒技術，能提供優異的空間解析度與穿透力。在操作模式方面，系統支援標準的 2D、M-Mode、彩色都卜勒、能量都卜勒 (S-Flow)、脈衝都卜勒 (PW) 及連續都卜勒 (CW)。此外，該系統整合了高階 3D/4D 立體成像技術、彈性成像 (ELASTOSCAN+)、組織都卜勒 (TDI/TDW) 以及專用於血管與小器官微流動影像的 MV-FLOW 模式。其臨床應用之廣泛，對於不同解剖結構與生理動態的擷取具有高度依賴性，因此其聲學輸出功率之控制與量測精度至關重要。</span>

<span style="color:#F87171">在本案中，新型探頭 CMV1-10 與 CMV1-10Z 為審查重點。這類探頭涉及高頻掃描與特定臨床場景（如小兒科或淺表組織），其中心頻率、軸向解析度與側向解析度的數值直接影響診斷結果的可靠性。目前資料中缺乏這些關鍵物理參數的量化數據，將對審查造成實質障礙。</span>

三、 缺失深度分析 (Deep-dive Analysis of Deficiencies)
1. 超音波特定安全標準 (IEC 60601-2-37) 之缺失
<span style="color:#F87171">目前申請人未檢附主機與搭配探頭之 IEC 60601-2-37 測試評估報告。此項缺失極為嚴重，因為該標準規範了診斷用超音波設備在熱指數 (TI) 與機械指數 (MI) 方面的安全上限。診斷用超音波對組織的熱效應與機械空化效應必須受到嚴格監控，特別是在產科應用中，胎兒組織對熱能極為敏感。若無此項報告，無法確認系統在最高聲學輸出操作模式下是否具備自動功率限制機制，亦無法評估其聲學輸出顯示值的準確度。</span>

2. 生物相容性評估之不完整性 (ISO 10993-1)
<span style="color:#F87171">探頭作為直接接觸患者皮膚、黏膜甚至手術中內臟組織的組件，其生物相容性是保障患者安全的第一道防線。審查發現，CMV1-10 等探頭未提供 ISO 10993-1 規範下的細胞毒性、致敏性與刺激性試驗報告。針對經陰道或經直腸等需接觸黏膜的探頭，原廠必須進一步提供高層次消毒 (HLD) 的化學相容性證明，以確保探頭材料在多次消毒循環後不會降解或釋放有害物質。</span>

3. AI/ML 技術文件之顯著缺口 (CADe/CADx)
<span style="color:#F87171">本案產品搭載了多項 AI 功能（如 ViewAssist、HeartAssist），這些功能被歸類為電腦輔助偵測或診斷工具。依照 TFDA 指引，原廠必須揭露：

演算法架構細節：例如模型是否採用卷積神經網路 (CNN)，其層數設定及訓練框架（如 PyTorch 或 TensorFlow）。
數據集異質性：目前的資料未能證明訓練數據包含了不同人種、年齡與性別的患者，這可能導致模型在台灣人群應用時出現偏見。
數據獨立性證明：必須明確描述測試數據（Test Data）如何與訓練數據絕對隔離，以排除過擬合 (Overfitting) 的風險。</span>

五、 AI CADe/CADx 專項審查重點
<span style="color:#F87171">針對 HERA Z20 所搭載的「ViewAssist」與「HeartAssist」功能，審查團隊必須嚴格把關其輸出邏輯。ViewAssist 作為一個自動視圖辨識系統，其核心挑戰在於「泛化能力」。超音波影像具有極高的噪聲與變異性，受操作者手法影響巨大。原廠必須提交「對比分析」，比較訓練數據與真實臨床環境中的影像差異，並說明輸入數據如何代表各種複雜的臨床狀況（如羊水過少、胎位不正等）。</span>

<span style="color:#F87171">此外，AI 模型具有「黑盒性」，其預測結果並非絕對。審查要求摘要中應包含對預測結果的「統計信心水準 (Statistical Confidence Level)」描述。若 ViewAssist 自動擷取的平面被模型判定為低信心度，應有明確的視覺化提示（例如顏色警示或文字提醒），告知醫師該自動建議可能存在誤差。針對「模型更新維護機制」，申請人必須說明未來若有模型參數調整，是否會進行再確效測試並向 TFDA 報備，以符合醫療器材持續學習的監管趨勢。</span>

六、 性能測試與品質控制分析
<span style="color:#F87171">在最終產品品質檢測方面，雖然原廠已檢附品質檢驗報告，但對於「測量功能」的驗證顯得薄弱。超音波診斷高度依賴於「距離（如胎兒雙頂徑）」、「體積（如心臟射血分數）」與「流速（如都卜勒血流）」的精確測量。申請人必須檢附本案所有臨床可使用測量項目的準確度規格。</span>

<span style="color:#F87171">針對 CMV1-10 探頭，性能測試資料應包含：

頻率範圍測試：驗證探頭是否能在宣稱的寬頻範圍內穩定運作。
掃描長度與角度驗證：確保實際掃描野與顯示屏上的刻度一致。
敏感度 (Sensitivity) 測試：在最大穿透深度下，是否仍能清晰辨識組織邊界。</span>
七、 結論與補件要求
<span style="color:#F87171">綜上所述，Samsung Medison HERA Z20 診斷用超音波影像系統雖然具備美國 FDA 上市許可，但在台灣查驗登記的技術文件準備上仍有顯著缺失。為了確保臨床使用的安全性與有效性，申請人必須在限定時間內完成以下資料補充：</span>

<span style="color:#F87171">安規補件：請檢附本案擬申請主機及搭配所有探頭之 IEC 60601-2-37 完整測試評估報告。</span>
<span style="color:#F87171">生物相容性補件：請提供所有探頭（尤其是 CMV1-10 系列）之 ISO 10993-1 生物相容性評估與試驗報告。</span>
<span style="color:#F87171">AI 技術文件補件：請依照《人工智慧/機器學習技術之電腦輔助偵測(CADe)及電腦輔助診斷(CADx)醫療器材查驗登記技術指引》，提供演算法架構、數據集來源、數據獨立性證明、獨立性能評估（AUC, ROC）及模型更新維護程序。</span>
<span style="color:#F87171">性能規格補件：請提供所有探頭之軸向與側向解析度實測值，以及所有臨床測量項目（距離、體積、流速等）的準確度規格資料。</span>
<span style="color:#F87171">測量功能補件：檢附具有定量/半定量量測能力之特殊軟體功能（如 IOTA-ADNEX）之演算法有效性評估報告。</span>
<span style="color:#F87171">本案審查意見將於補件資料送達後再次評估。若申請人無法提供上述關鍵數據，將建議不予通過查驗登記，以維護國人醫療品質。</span>
